# 一级标题

段落之间用空行分隔。

第二段落。_斜体_、**粗体**和`等宽字体`。无序列表
如下：

- 这一项
- 那一项
- 另一项

注意 --- 不考虑星号 --- 实际文本
内容从第 4 列开始。

> 引用块是这样
> 写的。
>
> 它们可以跨越多个段落，
> 如果你愿意的话。

使用 3 个破折号表示破折号。使用 2 个破折号表示范围（例如，"都在第 12--14 章"）。三个点 ... 将被转换为省略号。
支持 Unicode。☺

## 二级标题

这是一个有序列表：

1. 第一项
2. 第二项
3. 第三项

再次注意实际文本如何从第 4 列开始（距左侧 4 个字符）。这是一个代码示例：

    # 让我重申一下 ...
    for i in 1 .. 10 { do-something(i) }

正如您可能猜到的，缩进 4 个空格。顺便说一句，如果您愿意，
可以使用分隔块代替缩进块：

```
define foobar() {
    print "欢迎来到风味之国！";
}
```

（这使复制和粘贴更容易）。您可以选择标记分隔块，让 Pandoc 对其进行语法高亮：

```python
import time
# 快，数到十！
for i in range(10):
    # （但不要*太*快）
    time.sleep(0.5)
    print i
```

### 三级标题

    - 扁豆

2. 烧一些水。

3. 把所有东西倒进锅里，然后按照
    这个算法操作：

        找到木勺
        揭开锅盖
        搅拌
        盖上锅盖
